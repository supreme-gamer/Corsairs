Sails
{
Паруса - показывают текущее и максимальное количество повреждений, которые могут выдержать паруса.

Speed
{
Скорость - показывает максимальную скорость корабля на данный момент с учётом всех факторов (груз, умения капитана, повреждения кораблю) и максимально достижимую скорость корабля.
}

Maneuver
{
Манёвренность - показывает манёвренность корабля на данный момент с учётом всех факторов (груз, умения капитана, повреждения кораблю) и максимально достижимую манёвренность корабля.
}

AgainstWind
{
Наилучший курс судна относительно ветра, по достижении которого судно может развить максимальную скорость. Значения выражены в градусах.
Второе значение показывает разницу между текущим направлением ветра и наилучшим курсовым углом для данного судна.
}

Capacity
{
Дедвейт - показано текущее и максимальное значение вместимости корабля. Чем больше загружен корабль, тем он становится неповоротливее и тихоходнее.
Орудия, установленные на корабле, также учитываются при вычислении дедвейта. Их можно снять с корабля, освободив тем самым дополнительное пространство для груза. Для этого нужно в интерфейсе "Корабль" выбрать строку с текущими орудиями в списке параметров корабля.
}

Crew

}

Crew_Exp {Опыт команды}

Crew_Exp_hint
{
Опыт команды делится на три категории: морскую - влияет на управляемость кораблём, канонирскую - скорость перезарядки и точность наведения орудий, и абордажную - эффективность команды в рукопашных схватках. 
Опытная команда на корабле в меньшем составе может быть более эффективна, чем перегруженный салагами корабль.
Опыт во всех трёх категориях приобретается командой во время выполнения соответствующих действий (маневрирование, стрельба, абордаж), но также и утрачивается, если приобретённые навыки длительное время не подкрепляются практикой. 
}

TakeAllGoods_hint
{
Кнопка "Перегрузить все товары" осуществляет перегрузку товаров на корабль персонажа в соответсвии с их усреднённой удельной стомостью. Сначала перегружаются наиболее ценные товары, затем менее ценные, и так далее до тех пор, пока не закончится свободное место в трюме.
}

GoodsKeys_hint
{
Клавиши стрелок "Влево" и "Вправо" на клавиатуре обеспечивают премещение товара в выбранной строке таблицы по одной пачке. Если нажимать клавиши стрелок, удерживая при этом клавишу Shift - товар будет перемещён в полном объёме.
Клик правой кнопкой мыши по стрелкам на строке с товаром в интерфейсе переносит по десять пачек товара за раз.
}

Shipyard_hint
{
На верфи можно покупать, ремонтировать или продавать корабли.
Герой не может продать свой корабль, если у него есть компаньоны. Нельзя также купить корабль компаньону, если у героя нет своего корабля.
Для того, что бы сменить корабль героя или компаньона, сохранив всё содержимое трюма корабля и его экипаж, нужно выбрать в списке кораблей эскадры героя (в левой части интерфейса) корабль, который будет заменён на новый, а в списке кораблей, продающихся на верфи (в правой части интерфейса) - желаемый, после чего нажать кнопку "Купить".
Для того, что бы купить новый корабль (назначить нового компаньона), следует перед покупкой выбрать в списке слева пустой слот.
}
